# 🏨 Hotel Booking Project

Hotel Booking Project is a modern full-stack web application built using Django and MySQL that enables users to browse available hotel rooms, make online bookings, and manage reservations through a secure and user-friendly interface. The application includes user authentication, booking management, an admin dashboard, and responsive design powered by Bootstrap.

This project demonstrates full-stack web development skills, secure authentication, database management, and CRUD operations using the Django framework.

---

## 🚀 Features

- 🔐 User Registration & Login
- 👤 Secure User Authentication
- 🏨 Browse Available Hotels & Rooms
- 📅 Online Room Booking
- 📋 View Booking History
- ✏️ Update Booking Details
- ❌ Cancel Bookings
- 🛠️ Admin Dashboard for Managing Rooms & Bookings
- 📱 Responsive Bootstrap Interface
- 💾 MySQL Database Integration

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Ishwaryambika92/Hotel-Booking-Project.git
```

Navigate to the project folder

```bash
cd Hotel-Booking-Project
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Apply Migrations

```bash
python manage.py migrate
```

Run Development Server

```bash
python manage.py runserver
```

Open your browser

```
http://127.0.0.1:8000/
```
